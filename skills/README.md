# Skills

This directory contains deterministic business workflows and the documentation structure for future skills.

## Active Workflows
- `onboarding.js`
- `quote.js`
- `promo.js`
- `calendar.js`

## Rules
- Keep workflow inputs explicit.
- Keep outputs bounded and deterministic.
- Do not bypass validation, permissions, or approved-folder execution rules.
- Document new skills in the catalog, contracts, and guardrails folders.
